# 🚀 COSMOS ACADÉMIE

Un jeu éducatif spatial qui couvre **tout le programme du collège français (6ᵉ → 3ᵉ)** : Français, Maths, Histoire-Géo, Sciences/SVT/Physique-Chimie, EMC et Technologie.

- 🪐 Leçons claires et vivantes, dans un ton de prof sympa, avec schémas
- 🎯 7 défis par chapitre + un « boss » final
- 📝 Contrôles entre les chapitres ; pour la 3ᵉ, brevets blancs et brevet final (DNB)
- ⭐ Étoiles, notes sur 20, énergie et grades (Cadet → Pilote → …)
- 📱 Une seule page HTML, sans installation — fonctionne hors ligne

## Utilisation
Ouvre simplement `index.html` dans un navigateur, ou visite la version en ligne (GitHub Pages).

> La progression (étoiles, notes) est enregistrée localement dans le navigateur de chaque élève.
